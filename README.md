## About

**Memora** is a modern **web3 SDK built with Next.js**. It provides developers with pre-built components, API routes, and integration tools to simplify building decentralized applications and managing blockchain interactions.

- **Web3 Ready:** wallet integration, token operations, transaction tracking  
- **Next.js Components:** ready-to-use, responsive, and performant  
- **Developer Friendly:** pre-configured SDK functions for quick DApp setup  
- **Modern Design:** minimal, mobile-first, and scalable  

---

## Features

- SDK integration for Memora services  
- Pre-built Next.js UI components  
- API Routes for blockchain operations  
- Video/photo conversion tools for Roblox & Web3 apps  
- Real-time data and dashboard components
